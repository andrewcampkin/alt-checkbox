What is this?

alt-checkbox is a nice and flexible alternative to a standard HTML checkbox.

It's a jQuery plugin that comes with the default css that can be customized. You can style it as you want by modifying the plugin's styles in the css file.

By default, it utilizes icon fonts that allows to scale the checkbox to any size and apply some exotic icons.

alt-checkbox is created based on the standard HTML checkbox and synchronized with it to reflect the proper checked state.

Features

  • Custom styling
  • Flexible scaling
  • Icon fonts for icons
  • Mouse as well as keyboard (tab and space) to check and uncheck the checkbox
  • Label scaling together with the checkbox
  • Outlining the icon for unchecked state
  • Unchecked, checked, hovered and focused states
  • Ability to remove alt-checkbox and get standard HTML checkbox with proper state
  • Know more features by reading the source code on GitHub

Methods

altCheckbox(options)
Creates alt-checkbox based on HTML checkbox and hides HTML checkbox.
altCheckbox("clear")
Removes alt-checkbox and shows its original HTML checkbox.

Options

iconClass
CSS class for icon font. Default value is "fa-check".
sizeClass
CSS class to apply some size styles. Default value is "medium".
customClass
Some custom CSS class that may be added to alt-checkbox. Default value is "".
labelSameSize
Is label the same size as alt-checkbox. Default value is true.
outlineUnchecked
Is icon outlined for the passive state. Default value is true.

How does it work?

  1. Include jQuery library to the head section
  2. @import the desired icon font in the jquery.alt-checkbox.icon-font.css and include this css file in the head section, Font Awesome is used by default
  3. Style the checkbox if you want in the jquery.alt-checkbox.css file
  4. Include jquery.alt-checkbox.css to the head section
  5. Include jquery.alt-checkbox.js to the head section
  6. Select some checkboxes with the jQuery and apply the alt-checkbox plugin for them specifying some options if you need
